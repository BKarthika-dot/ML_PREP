# Explainable AI (XAI)

This folder is dedicated to my study and implementation of **Explainable Artificial Intelligence (XAI)** techniques as part of my machine learning preparation.

The focus is on understanding **why** a model makes a prediction, not just achieving high accuracy. This is especially important for real-world ML systems used in high-stakes domains such as hiring, healthcare, and finance.

This folder will gradually evolve as I add implementations, experiments, and notes.
